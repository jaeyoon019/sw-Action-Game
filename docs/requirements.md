# 요구사항 정의서

## 기능 요구사항 (Functional Requirements)

| ID | 요구사항 내용 | 우선순위 |
|----|--------------|----------|
| FR-01 | 사용자는 입력 후 100ms 안에 캐릭터를 조작할 수 있어야 한다. | 상 |
| FR-02 | 사용자는 사망 시 즉시 체크포인트에서 재시작할 수 있어야 한다. | 상 |
| FR-03 | 사용자는 반복 플레이를 통해 자신의 조작 숙련도를 향상시킬 수 있어야 한다. | 상 |
| FR-04 | 사용자는 다양한 지형과 트랩을 포함한 스테이지를 플레이할 수 있어야 한다. | 중 |
| FR-05 | 사용자는 노 데미지 또는 제한시간 내 클리어와 같은 도전 과제를 수행할 수 있어야 한다. | 하 |

## 비기능 요구사항 (Non-Functional Requirements)

| ID | 요구사항 내용 | 우선순위 |
|----|--------------|----------|
| NFR-01 | 시스템은 Windows, macOS, Linux에서 원활히 실행될 수 있도록 최소 사양 기준에서 초당 60FPS 이상을 유지하며 메모리 사용량은 1GB 이하로 동작해야 한다. | 상 |
| NFR-02 | 시스템은 스테이지 클리어 시 명확한 보상을 제공해야 하며, 보상 획득 과정은 사용자에게 1초 내 인지될 수 있어야 한다. | 상 |
