# 에셋 출처 (Credits)

이 프로젝트에서 사용한 모든 외부 에셋의 출처와 라이선스를 기록한다.
**새 에셋을 추가할 때마다 아래 해당 표에 한 줄씩 누적**한다. 저작권 분쟁과 과제 감점을 예방하는 핵심 문서이므로, 코드보다 먼저 갱신할 것.

## 라이선스 약칭

| 약칭 | 의미 | 출처 표기 | 상업 이용 | 수정 / 2차 창작 |
|------|------|-----------|-----------|----------------|
| **CC0 1.0** | Public Domain | 불필요 (권장) | 가능 | 가능 |
| **CC-BY 4.0** | 출처표기 조건 | 필수 | 가능 | 가능 |
| **CC-BY-SA 4.0** | 출처표기 + 동일 라이선스 공유 | 필수 | 가능 | 가능 (동일 라이선스로) |
| **CC-BY-NC 4.0** | 출처표기 + 비상업 | 필수 | **불가** | 가능 |
| **MIT / Apache 2.0** | 코드 라이브러리용 | 라이선스 텍스트 포함 | 가능 | 가능 |
| **자체 제작** | 팀원 본인 제작 | — | — | — |

> 위 목록에 없는 라이선스는 **전체 명칭과 URL을 기재**한다.
> 라이선스가 명시되지 않은 에셋은 사용 금지 (출처 불명 = 저작권 위반 위험).

## 추천 출처 (라이선스가 명확한 곳)
